# 机械图纸旋转角度判断规则

## 适用范围

本规则用于判断机械图纸扫描件相对于正确阅读方向被顺时针旋转了多少度。

前提假设：

- 图纸符合机械制图规范。
- 标题栏在正确方向下应位于页面下方、底边满宽区域或右下方。
- 扫描件可能被顺时针旋转 0、90、180 或 270 度。

## 核心原则

优先定位标题栏，不优先依据零件图形方向判断。

零件视图本身可能横放或竖放，但标题栏位置具有规范约束，因此标题栏是判断页面方向的主依据。

## 旋转角度映射

根据当前页面中标题栏的位置，判断图纸已经被顺时针旋转的角度：

| 当前标题栏位置 | 已顺时针旋转角度 |
| --- | --- |
| 下方、底边满宽或右下方 | 0 度 |
| 左侧 | 90 度 |
| 上方或左上方 | 180 度 |
| 右侧或右上方 | 270 度 |

## 校正角度映射

若需要把图纸校正到标题栏位于下方或右下方，可按下表旋转当前页面：

| 当前标题栏位置 | 校正操作 |
| --- | --- |
| 下方、底边满宽或右下方 | 不旋转 |
| 左侧 | 逆时针 90 度，或顺时针 270 度 |
| 上方或左上方 | 旋转 180 度 |
| 右侧或右上方 | 顺时针 90 度，或逆时针 270 度 |

## 判断细则

1. 标题栏可能是页面右下角的表格，也可能沿底边横向展开。
   竖向纸张绘制的机械图纸中，标题栏可以位于页面下方并刚好占满图纸宽度；此时应判为下方或底边标题栏，而不是强行归入左下角或右下角。
2. 标题栏中通常包含图号或图样代号、图名或名称、材料、比例、日期、单位名称、签字栏等字段。
3. 若标题栏在右侧，不应直接判为 90 度；按几何映射，它通常表示图纸已顺时针旋转 270 度。
4. 若标题栏在左侧，通常表示图纸已顺时针旋转 90 度。
5. 若标题栏在顶部，通常表示图纸已顺时针旋转 180 度。
6. 若标题栏已经在底部或右下方，通常表示图纸方向正确，即 0 度。
7. 当标题栏位置不完整或边缘被裁切时，可用标题栏文字阅读方向、表格结构和图框边界辅助判断。
8. 真正标题栏必须贴着图纸外框线，标题栏外边界和图纸周围框线之间不应存在空隙；仅靠近边缘但没有贴住图框线的候选，应降权或进入人工复核。
9. 位于图纸主体内部、与外框线存在明显间隔的表格状零件或局部结构，不应作为标题栏；这是区分长得像标题栏的零件和真正标题栏的重要规则。
10. 标题栏字段应按组合判断，而不是按单词命中判断。较强组合通常包括两类字段同时出现：
   - 人员与流程字段：设计、制图、校对、工艺、标准或标准化、审核、批准、日期。
   - 图纸属性字段：图名或名称、图号或图样代号、材料、比例、重量、表面积、单位。
11. 只出现 `材料`、`日期`、`审核` 等单个词，不能判定为标题栏；明细栏、技术要求表和局部说明表也可能包含少量相似词。
12. 标题栏内部表格通常不是完全均匀网格，而是由签字/日期小格、图名/图号/材料/比例等不同尺寸区域组合而成。格子大小不一、行列不规则可作为辅助证据，但不能单独替代贴图框线和字段组合。
13. 多个相连表格候选中，只有当人工复核或字段/结构证据能确认其属于标题栏主体时，才可视为可接受标题栏候选；图纸中的其他表格即使贴近标题栏或与标题栏方向相似，也应作为非标题栏候选降权、拒绝或进入人工复核。
14. 标题栏 OBB 框存在小角度偏差时，应区分可容忍定位误差和结构性误检。若候选框仍覆盖真实标题栏主体，约 3 度级别的整体角度偏差可视为可接受；若候选框落在其他表格或零件区域，则不能因角度或贴边相似而放行。
15. 普通表格误检与页面方向无关。由于扫描件可能被旋转，非标题栏表格可能出现在上、下、左、右任意方向；自动流程不得把某一方向上的表格误检当作特殊个案处理。

## 自动化实现建议

后续开发自动工具时，建议流程如下：

1. 将 PDF 页面渲染为图像。
2. 检测图框和标题栏候选区域。
3. 优先寻找表格密集区域、固定字段文字和右下/底边布局特征。
4. 判断标题栏候选区域位于页面的下、左、上、右哪一侧；若标题栏横跨边缘，应输出边位置，而不是强制压成角位置。
5. 按本规则的映射表输出已旋转角度和校正角度。
6. 对低置信度结果保留人工复核标记。
7. 将候选框是否贴住图纸外框线作为强证据；若扫描裁切或图框线缺失导致无法确认贴边关系，应输出低置信或人工复核状态。
8. OCR 或 VLM 兜底应统计字段簇组合，而不是单词命中数；字段证据应服务于候选真实性确认，不应单独覆盖图框贴边和位置证据。
9. 对候选内部网格可计算单元格大小不均匀度、签字小格和大信息格混合程度，用作区分标题栏与均匀零件表格的辅助证据。

## 输出建议

自动工具输出时建议同时给出：

- 文件名或页码。
- 标题栏当前位置。
- 判断出的已顺时针旋转角度。
- 建议校正角度。
- 置信度或是否需要人工复核。

标题栏位置字段建议至少支持：

- `bottom_edge`：标题栏沿页面底边横向展开，可能占满或接近占满图纸宽度。
- `bottom_right`：标题栏集中在右下角。
- `left_edge`：标题栏沿左边展开。
- `top_edge`：标题栏沿顶部展开。
- `right_edge`：标题栏沿右边展开。
- `no_title_block`：页面没有可确认标题栏，例如只有零件表格、明细表或说明表。
